# 短视频特效镜头库

这是一个可本地编辑、可静态部署的短视频特效镜头库执行网站，用于团队查看和沉淀特效镜头案例。

## 在线查看

如果部署到 GitHub Pages，打开仓库的 Pages 链接即可查看。

## 本地打开

直接双击 `index.html` 可查看页面。

如需用本地服务预览：

```bash
npm start
```

然后打开：

```text
http://localhost:3000
```

## 内容结构

- `index.html`：网页主体
- `style.css`：页面样式
- `script.js`：镜头库数据和交互逻辑
- `01_方案/`：每条参考视频的解析文档
- `03_视觉参考/`：参考视频和关键帧

## 注意

当前版本适合团队通过链接查看。如果需要多人在线共同编辑，需要再接入后端数据库或表格同步方案。
