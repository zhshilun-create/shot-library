## 镜头概述

这条视频把“视频行业”包装成一款游戏。开头是霓虹城市背景上的登录界面，随后进入角色管理、删除角色确认、职业选择、捏脸自定义、装备选择，最后以 Loading 画面进入游戏世界。它的特效核心不是单个爆点，而是一整套“游戏化 UI 叙事”：把现实人物、职业身份、器材工具和视频行业梗变成游戏菜单、职业栏、属性条和装备栏。

## 执行网站卡片文案

### 特效名称

视频行业游戏化角色创建

### 适用场景

团队介绍、职业介绍、视频行业创意短片、品牌活动开场、设备/工具展示、栏目包装、达人账号世界观设定、招聘宣传。

### 视觉钩子

把现实视频行业拍成一款游戏：登录界面进入角色，选择摄影师/剪辑师/导演等职业，再用捏脸滑杆调整人物，最后选择相机、稳定器、话筒等装备。观众会被“现实职业游戏化”的设定吸引，信息量大但形式好懂。

### 前期怎么拍

选择夜景、街区、天台、停车场、片场等有氛围的场景，最好有霓虹灯、路灯、湿地反光或工业背景。人物拍摄要按游戏界面需求预留空间：登录界面人物可站远一点，职业选择需要多人横排，捏脸界面需要单人半侧脸特写，装备选择需要器材单独拍干净素材。拍摄时多留画面上方和两侧，用来叠 UI。

### 需要素材

夜景城市背景、角色站立素材、多职业人物横排素材、单人脸部特写、器材道具素材、低机位过场、桥梁/城市 Loading 背景、游戏 UI 元素、发光文字、属性条、选择框、按钮音效。

### 后期制作方法

先确定游戏流程：登录、角色管理、职业选择、捏脸、装备选择、加载进入。每一段用统一的 UI 语言，比如青绿色霓虹字、半透明面板、扫描线、发光边框、按钮提示。人物和装备不需要复杂抠像，但要用发光轮廓、地面光圈、选择框和属性条增强游戏感。UI 动画建议用淡入、滑动、闪烁、进度条填充和选项切换，配合按键音、确认音和加载音。

### AI 提示词

将现实视频行业创意短片包装成游戏角色创建界面，夜晚霓虹城市街区背景，一组视频行业从业者横排站立，画面上叠加青绿色发光游戏UI，包含登录界面、选择职业、捏脸滑杆、选择摄影器材装备、属性条和加载界面，真实电影质感，赛博朋克但不过度夸张，半透明HUD界面，竖屏9:16，清晰可读的界面布局，不要水印，不要乱码文字，不要卡通。

### 负面提示词

不要过度科幻机甲，不要复杂到看不清人物，不要 UI 遮挡脸部，不要乱码文字，不要装备变形，不要低清晰度，不要颜色过曝，不要人物比例异常。

### 难度

高。难点在于它是系统化包装，不是单一特效。需要统一 UI 风格、设计多个界面状态，还要让人物、职业、装备和属性条之间有逻辑关系。

### 可复现要点

1. 先写清楚游戏流程，不要边剪边想 UI。
2. 所有 UI 使用同一套颜色、字体、发光强度和按钮样式。
3. 人物拍摄必须给 UI 留空间，尤其是顶部标题、左侧菜单和底部按钮。
4. 职业选择段要横排站位，装备选择段要单独拍道具。
5. UI 不要只贴静态图，要有进度条、选中框、闪烁、确认音等交互反馈。

## 推荐拆分

- 0-5 秒：游戏登录界面，建立概念。
- 5-10 秒：角色管理/删除确认，加入幽默设定。
- 10-18 秒：选择职业，展示视频行业身份。
- 18-26 秒：捏脸界面，展示人物特写和滑杆。
- 26-35 秒：选择装备，把器材变成游戏道具。
- 35-39 秒：Loading 进入世界，作为收尾或转场。
