# 镜头库上线协作说明

## 先在局域网跑起来

在项目目录执行：

```bash
npm start
```

启动后，本机访问：

```text
http://localhost:4173
```

同一个局域网里的同事访问：

```text
http://你的电脑IP:4173
```

打开后页面左上角会显示“团队在线模式”。这个模式下，文字修改和上传的视频/图片会保存到项目目录里的 `data/` 和 `uploads/`。

## 上线到云服务器

把整个 `镜头库` 文件夹上传到服务器，安装 Node.js 后执行：

```bash
npm start
```

如果要改端口：

```bash
PORT=8080 npm start
```

再用 Nginx、宝塔、1Panel 或服务器防火墙把对应端口开放给团队访问。

## 数据位置

- 镜头库文字数据：`data/state.json`
- 团队上传素材：`uploads/`
- 本地参考素材：`03_视觉参考/`

上线前建议定期备份这三个位置。

## 注意

现在是轻量协作版，没有账号登录和权限区分。适合团队内部使用，建议先放在公司内网、VPN 或有访问控制的服务器后面。
